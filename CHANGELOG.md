# Changelog

## v1.0.0 — 2026-01-31

Initial release.

- Album art display with play/pause overlay
- Now Playing Strip with configurable display modes (Artist, Album, Track, Time)
- Spanning progress bar across 1–4 touch strip panels
- Dynamic color extraction from album art
- Play/Pause, Previous, Next buttons with hold-to-seek
- Track Info display (codec, bitrate, track count)
- Time Elapsed display with progress bar
- Real-time interpolated time updates (200ms refresh)
- Configurable sync offset for Plex reporting delay compensation
- Built-in connection test
